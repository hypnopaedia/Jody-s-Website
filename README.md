# Introduction
Personal, MySpace-style app made with the following stack:
    - React
    - TypeScript
    - Redux (Toolkit)
    - React Router
    - Axios
    - Sass/Scss
    - Bootstrap

Other libraries used:
    - Semantic UI
    - Dayjs
    - react-markdown

Enjoy!
